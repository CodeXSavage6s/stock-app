# Stock Market Tracker

A Next.js TypeScript application that provides a searchable interface for viewing detailed stock information. Users can search for ticker symbols, view real‑time charts via TradingView widgets, and add stocks to a watchlist.

## Features
- **Search** – Quick search command (Ctrl/Cmd + K) to navigate to a stock detail page.
- **Dynamic Stock Pages** – Each stock has its own page (`/stocks/[symbol]`) showing charts, company info and key metrics.
- **Watchlist** – Add or remove stocks from a personal watchlist.
- **TradingView Integration** – Interactive charts powered by TradingView.

## Getting Started
```bash
npm install
npm run dev   # runs the Next.js development server
```

Visit `http://localhost:3000` and use the search button or shortcut to explore stocks.
